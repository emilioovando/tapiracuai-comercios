# Politica de privacidad - Tapiracuai Comercios

Ultima actualizacion: 29 de junio de 2026

## 1. Datos que podemos recopilar
Segun el uso de la plataforma, podemos recopilar:
- Nombre y apellido.
- Telefono o WhatsApp.
- Correo electronico.
- Barrio, zona o direccion de entrega.
- Datos del comercio y responsable.
- Logos, fotos y productos cargados.
- Pedidos, consultas, disponibilidad y mensajes relacionados.
- Datos tecnicos basicos, como dispositivo, navegador, fecha, hora y actividad dentro de la app.
- Ubicacion aproximada o precisa solo si el usuario la autoriza para funciones de mapa o delivery.

## 2. Para que usamos los datos
Usamos los datos para:
- Mostrar comercios y productos.
- Conectar clientes con comercios.
- Enviar consultas o pedidos por WhatsApp.
- Administrar registros de comercios.
- Aprobar, destacar o suspender comercios.
- Mostrar disponibilidad, precios y estados de pedido.
- Mejorar seguridad, funcionamiento y experiencia de la app.
- Enviar avisos importantes sobre la plataforma.

## 3. Datos compartidos con comercios
Cuando el cliente arma un pedido o consulta, se puede compartir con el comercio: nombre, barrio, direccion si corresponde, productos, cantidades, total estimado, nota y canal de contacto.

## 4. Datos compartidos con terceros
Podemos usar servicios externos para hosting, base de datos, analitica, mapas, autenticacion, correo, WhatsApp, Meta/Facebook/Instagram o herramientas similares. Solo se deberian compartir los datos necesarios para operar la plataforma.

## 5. Fotos, logos e imagenes
Las imagenes cargadas por clientes o comercios se usan para mostrar perfiles, comercios, productos o comprobantes dentro de la app. El usuario o comercio debe tener derecho a usar esas imagenes.

## 6. Ubicacion
La ubicacion se usara solo si el usuario la autoriza. Puede servir para mapas, zonas de delivery, comercios cercanos o seguimiento de pedido. En una primera etapa, el seguimiento puede ser solo estimado y no en tiempo real.

## 7. Seguridad
Se deben aplicar medidas razonables para proteger cuentas, contrasenas, datos de comercios, pedidos y datos personales. Las contrasenas nunca deben guardarse en texto visible; deben almacenarse cifradas o mediante un proveedor seguro de autenticacion.

## 8. Conservacion de datos
Los datos se conservaran mientras sean necesarios para operar la cuenta, pedidos, historial, soporte, cumplimiento legal o seguridad. El usuario podra solicitar eliminacion o correccion de sus datos por el canal oficial.

## 9. Derechos del usuario
El usuario puede solicitar acceso, correccion, actualizacion o eliminacion de sus datos, sujeto a verificaciones razonables de identidad y a obligaciones legales o comerciales aplicables.

## 10. Menores de edad
La plataforma no deberia estar dirigida a menores sin autorizacion de sus padres o responsables. Antes del lanzamiento se debera definir una regla clara de edad minima o uso supervisado.

## 11. Contacto de privacidad
Antes del lanzamiento se debe publicar un correo o WhatsApp oficial para solicitudes relacionadas con privacidad y datos personales.
